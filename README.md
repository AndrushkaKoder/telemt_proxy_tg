Compose файл + toml конфиг прокси для telegram. 
Работает стабильно на 17.03.2026

Алгоритм подключения:
1) Сгенерировать secret: openssl rand -hex 16
2) Подставить домен ру сайта для маскировки от active probing (РКН может открыть прокси своим парсером, увидеть, что там не сайт и заблочиь ip)
3) запустить докер
4) в логе контейнера взять ссылку вида tg:xxxx, заменить 127.0.0.1 на реальный ip сервера
